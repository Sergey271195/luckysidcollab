Lavanda$_Bot
Доступны следующие команды
/start - Запускает бота   <-------- done
/help - Список доступных команд  <-------- done
/add - добавляет данные пользователя  <-------- done
/stat - статистика пользователя  <-------- done
/datavisualization - Таблицы и графики  <-------- done
Принцип работы бота. 
Пользователь запускает бота командой /start, и видит перед собой приветственное сообщение “Здравствуйте, команда Lavanda$_Bot 
приветствует вас! Наш бот поможет вам вести статистику ваших 
расходов и выдавать вам её в виде удобных графиков.”
Выбирая команду /add пользователю предлагаются категории трат (Продукты/Услуги/Техника/АЗС/Рестораны). Пользователь выбирает категорию, пишет потраченную сумму и сумма добавляется в ДБ. 
Выбирая команду /stat, пользователю выдается статистика его расходов.
Выводится сообщение:
“За прошедшую неделю вами было потрачено:
 x рублей.
Самые крупные категории затрат:
1.Еда a рублей
2.АЗС k рублей
3.Услуги t рублей
За месяц : g рублей
За год: l рублей”
Datavisualization - Пользователю предлагаются несколько вариантов визуализации данных. Предлагаю 1. Круговая остается без изменений 2. График расходов за год (по месяцам). Возможное крупное изменение - ввести вывод статистики за неделю/месяц/год. Соответственно круговая - все просто, а график будет выводится по месяцам, неделям и дням.
